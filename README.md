# Olist E-Commerce EDA
This project focuses on Exploratory Data Analysis (EDA) for the Olist Brazilian E-Commerce dataset. The analysis aims to discover business insights related to customer behavior, order trends, payments, product categories, delivery performance, and seller activity.

The project was built using Python inside Jupyter Notebook and includes data cleaning, preprocessing, visualization, and analytical exploration.

---

## Dataset
The dataset contains real-world Brazilian e-commerce data from Olist, including:

- Customers
- Orders
- Order Items
- Payments
- Reviews
- Sellers
- Products
- Product Categories

---

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

---

## Project Workflow
1. Data Loading
2. Data Cleaning
3. Date Conversion & Formatting
4. Exploratory Data Analysis
5. Business Questions & Insights
6. Data Visualization

## Key Insights
